Simulador de Administración de Socios

Descripción del proyecto

Este simulador representa un sistema básico de administración de socios de una institución. Permite registrar nuevos socios, visualizar la lista de socios y gestionar el cobro de cuotas mensuales.

La interacción del usuario se realiza mediante cuadros de diálogo de JavaScript (Prompt, Alert y Confirm) y los resultados se visualizan en la consola del navegador. El objetivo del proyecto es aplicar variables, arrays, funciones, ciclos y condicionales para simular la lógica de un sistema administrativo.

Abra la consola del navegador (F12) para interactuar con el simulador.
